# Pasta de imagens

Coloque aqui as imagens do site, por exemplo:

- `hero.jpg` — imagem de fundo ou destaque da seção principal
- `salao.jpg` — foto do salão (seção Sobre)
- `logo.png` — logo do salão (se usar no header)
- ícones ou fotos dos serviços, se desejar

Formatos recomendados: JPG ou PNG, otimizados para web.
